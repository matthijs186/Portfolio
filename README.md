# portfolio

## Get started

1. Install modules.

   ```
   npm i gulp-cli -g
   npm i
   ```

2. Run Gulp.

   ```
   gulp
   ```


## Gulp tasks

Run each task with `gulp <task>`.

- `sass` - Compile Sass files in `src/assets/sass/`.
- `sass:watch` - Watch Sass files for changes in `src/assets/sass/`.
