# portfolio

## Get started

1. Install modules.

   ```
   npm i grunt-cli -g
   npm i
   ```

2. Run Grunt.

   ```
   grunt
   ```


## Grunt tasks

Run each task with `grunt <task>`.

- `mkdir` - Create the `src/assets/css` directory.
- `sass` - Compile Sass files in `src/assets/sass/`.
- `watch` - Watch Sass files for changes in `src/assets/sass/`.
